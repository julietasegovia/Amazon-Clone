# Amazon Clone
### Why did I do this?
After doing the Spotify Clone (https://github.com/julietasegovia/Spotify-Clone) I wanted to create something similar with a more complex backend. I landed on Amazon because it has a very simple aesthetic and easy to replicate front-end, and with a log in, a shopping cart, a search bar and more, it was a pretty decent back-end challenge.
### How did I do this?
Just like in my previous clone, I used React + Vite and Tailwind for dynamic styling. The first hours of this project were spent mostly on styling. I also used claude to help me debug.
